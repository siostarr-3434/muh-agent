# Güvenlik bildirimi

Bu depo kişisel ve hassas iş akışları için hazırlanır. Güvenlik açığını herkese açık issue olarak paylaşmayın. Depo sahibine GitHub üzerinden özel güvenlik bildirimi gönderin.

Bir bildirime şunları ekleyin: etkilenen dosya/akış, yeniden üretme adımları, olası etki ve varsa güvenli düzeltme önerisi. Gerçek BSN, token, belge, e-posta gövdesi veya başka kişisel veri eklemeyin.

Aktif olarak desteklenen sürüm `main` dalındaki son yayındır. Para transferi, DigiD otomasyonu veya onaysız dış gönderim desteklenen özellik değildir.
